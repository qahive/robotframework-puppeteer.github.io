Async Await Feature

Thursday, Jul 12, 2018 by Atthaboon S.

RF Puppeteer allow to interact with UI Element using Run Async Keywords keyword. Run Async Keywords allow to combine multiple synchronous keywords and run all keywords at the same time (asynchronous). Example: Login with valid password will redirect to login welcome page. Run Async Keywords on line 11 that cover keywords line 12 - 13. This will make keyword Wait For New Window Open and Click Element run as asynchronous.
